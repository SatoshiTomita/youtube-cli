# youtube-cli

![MIT License](https://img.shields.io/apm/l/atomic-design-ui.svg?)

A simple CLI application for youtube.

## Requirements

- [node](https://nodejs.org/en/)
- [mpv](https://mpv.io/)
- [youtube-dl](https://youtube-dl.org/)

## Installation

    $ cd /path/to/folder
    $ npm i -g .


## Usage

    $ ytcli
